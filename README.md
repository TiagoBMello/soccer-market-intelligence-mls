⚽ MLS 2025 – Football Market Intelligence
Sobre o Projeto

Este projeto simula a atuação de um departamento de análise de mercado no futebol profissional, utilizando dados da MLS 2025 para identificar jogadores com alto desempenho e potencial subvalorização.

Integra engenharia de dados, modelagem relacional, análise estatística e visualização executiva.
